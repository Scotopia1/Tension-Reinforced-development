# TRD Group — Immersive Site

Next.js 16 + React 19 + Tailwind 4 + GSAP 3.14 + React Three Fiber.
Cinematic scroll-driven experience for TRD Group (Tension Reinforced Developments).

See `AGENTS.md` for collaboration rules and `docs/plans/2026-04-16-trd-immersive-design.md`
for the architectural design.

## Develop

```
npm install
npm run frames   # one-time: builds public/frames/hero from ../TRD-content
npm run dev
```

Open <http://localhost:3000>.

## Build

```
npm run build
npm start
```

## Lint

```
npm run lint
```
